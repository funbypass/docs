## Base URL

All API requests are made to:

```
https://api.funbypass.com
```

## How It Works

<Steps>
  <Step title="Create a Task">
    Send a `POST /createTask` request with your API key, site key, proxy, and optional parameters.
  </Step>
  <Step title="Poll for Status">
    Use `GET /getTaskResult/{taskId}` to check the task status while it remains `processing`.
  </Step>
  <Step title="Retrieve the Result Token">
    Once the status is `ready`, the response includes the result token for that session.
  </Step>
</Steps>
